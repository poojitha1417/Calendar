📅 Wall Calendar App
A beautiful, interactive wall calendar built with React and Vite. Features unique monthly themes with stunning nature photography, holiday highlights, and a clean modern UI.

✨ Features

🎨 12 Unique Monthly Themes — Each month has its own color palette and background image (Winter Peaks, Golden Hour, Ocean Breath, and more)
🗓️ Interactive Calendar Grid — Navigate between months with smooth transitions
🎉 Holiday Highlights — Major holidays marked automatically (New Year's, Valentine's Day, Halloween, Christmas, and more)
📌 Today Indicator — Current date is always highlighted
⚡ Fast & Lightweight — Built with Vite for instant load times
📱 Responsive Design — Works on desktop and mobile

🚀 Getting Started
Prerequisites

Node.js (v16 or higher)
npm

Installation
bash# Clone the repository
git clone https://github.com/poojitha1417/Calendar.git

# Navigate into the project
cd Calendar

# Install dependencies
npm install

# Start development server
npm run dev
Open your browser and go to http://localhost:5173

🏗️ Build for Production
bashnpm run build
The output will be in the dist/ folder.

☁️ Deployment (Vercel)
This project is configured for Vercel deployment via vercel.json.

Push your code to GitHub
Go to vercel.com and import your repository
Vercel will auto-detect Vite as the framework
Click Deploy — done! 🎉

Live URL format: https://calendar-xxx.vercel.app
